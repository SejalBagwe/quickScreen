Your Question:

Write a python code to generate a hash which will be validated by a server. The hash will be based on the current epoch time. (epoch time is just a simple way computers use to represent current time as a single float variable. It is the number of seconds that have passed since 1st January, 1970).

Method to do:

You can print the current epoch time in python by writing the following code:
import time
print(time.time())
# Output : {% if language == 'python' %} 1473636383.373736 {% else %} 1473636383.373 {% endif %}

First calculate the sum of all individual digits before the fullstop as sum1. Then calculate the sum of all individual digits after the fullstop as sum2.
Your hash final hash output will be:
(the current epoch time)-sum1-sum2.

Expected Output:

That is, your generated hash output should look something like this:
1473636383.373736-44-29
